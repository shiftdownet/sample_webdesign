文章表示 - text(テキスト),msg(メッセージ)

説明 

text(またはmsg)
	表示したい文章(何行でも可能)
endtext(または「.」)

text(またはmsg) 番号
	表示したい文章(何行でも可能)
endtext(または「.」)

text(またはmsg) 番号 X座標 Y座標 横幅 行数 文字色 背景色 枠線色 配置
	表示したい文章(何行でも可能)
endtext(または「.」)
「text」の次の行から「endtext」の行の前までを表示し、クリックされるまで待つ。
「msg」の次の行から「msgtext」の行の前までを表示する。クリック待ちはしない。

注意

半角の「,」「%」「&」「|」等は全角に変換されます。 また、一部の文字は表示されないことがあります

パラメータ

行数
表示する行数を指定します。
省略した場合、デフォルトでは3行です。
0を指定すると自動的に行数を指定します。
番号
1～9の番号を指定します。
通常のtextとあわせて最大10個まで表示可能です。
1～9の範囲外に指定すると1とみなされます。
X座標,Y座標,横幅
画面左上が(0,0)で右下に向かって正となります。
横：グリッドの目盛り+275、縦：グリッドの目盛り+200
しかし、コマンドの前に「_(アンダースコア)」をつけるとグリッドのメモリと一致します。
例：_text　_msg
横幅は指定した値より15ほど大きくなります。
文字色 背景色 枠線色
16進数6桁で指定します。
変数の場合は10進数で指定します。
背景色または枠色に-1を指定すると表示しません。
配置
「center」「left」「right」のどれかを指定します。
center：中央に表示します。
left:左寄せで表示します。
right:右寄せで表示します。

例

通常の方法で文章を表示
text
	こんにちは。
	今日の天気はいかがでしょう。
endtext
行数を指定して文章を表示
text 2
	こんにちは。
	今日の天気はいかがでしょう。
endtext
複数のパラメータを指定して文章を表示
text 1 0 0 250 2 ffffff 000000 ffffff left
	こんにちは。
	今日の天気はいかがでしょう。
endtext
グリッドのメモリと一致する方法で表示
_text 1 0 0 250 2 ffffff 000000 ffffff left
	こんにちは。
	今日の天気はいかがでしょう。
endtext

文章消去 - clear(クリア)

説明 

clear 番号
指定した番号のtextやmsgを消去する。

パラメータ

番号
消去するテキスト番号を指定します。
値は0～9までの値が指定できます。(0は通常のテキストです。)
範囲外の値を指定した場合すべて消去します。
パラメータを省略した場合もすべて消去します。

例

表示したテキストを消去
テキストを表示します。
msg 1 0 0 200 2 ffffff 000000 ffffff left
	text番号1
endtext
msg 2 0 100 200 2 ffffff 000000 ffffff left
	text番号2
endtext
msg 3 0 200 200 2 ffffff 000000 ffffff left
	text番号3
endtext
番号1を消去します。
clear 1
続いて全てのテキストを消去します。
clear
完了しました。

文章振動 - shake(シェイク)

説明 

shake 番号
指定した番号のtextやmsgを振動する。

パラメータ

番号
振動するテキスト番号を指定します。
値は0～9までの値が指定できます。(0は通常のテキストです。)
範囲外の値を指定した場合すべて振動します。
パラメータを省略した場合もすべて振動します。

例

表示したテキストを振動
テキストを表示します。
msg 1 0 0 200 2 ffffff 000000 ffffff left
	text番号1
endtext
msg 2 0 100 200 2 ffffff 000000 ffffff left
	text番号2
endtext
msg 3 0 200 200 2 ffffff 000000 ffffff left
	text番号3
endtext
番号1を振動します。
shake 1
続いて全てのテキストを振動します。
shake
msg
	完了しました。
.
shake 0

コメント

説明 

; 文章
「;(セミコロン)」のうしろに何を書いても無視される。

例

コメント文の例
; コメント文はどこでもかけます。
text ; 命令の後ろでも書けます。
	こんにちは。 ; 命令の中でも書けます。
endtext
 

使用ブラウザはOperaを推奨します。 他ブザウザでは多少レイアウトが崩れる可能性があります。
